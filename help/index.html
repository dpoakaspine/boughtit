Install

Upload the wp-boughtit folder you extraced from your downloaded ZIP file into to the /wp-content/plugins/ folder of your current WordPress installation. After uploading the whole folder, there should be a new Plugin under Plugins called BoughtIt. Activate it.



Design

BoughtIt comes with everything you need to customize it to your site. Just go to your WordPress-Admin under your-site.com/wp-admin/. Go to Design => Customizer and wait until it has finished loading. There sould be a new tab called WP BoughtIt Settings. Click on it and you will find multiple options for designing your card.






Configuration

You can configure multiple options just below the design section. All options should be self-explanatory.





Documentation

We added methods to manipulate the main query. More options are coming soon.

boughit_order_query_args manipulates the main order query. BoughIt gets its product items from this main query and shuffles everything for randomness.

Example: 

function my_boughit_order_query_args( $query_args ) {
  //return the original query args
  return $query_args;
}

add_filter( 'boughit_order_query_args', 'my_boughit_order_query_args', 10, 3 );


boughit_customer_orders contains the main order query posts. BoughIt gets its product items from these objects and shuffles everything for randomness.

Example: 
function my_boughit_customer_orders( $orders ) {
//return the original orders
return $orders;
}

add_filter( 'boughit_customer_orders', 'my_boughit_customer_orders', 10, 3 );
